# IPClaw Topic Ideas Prompt（优质选题生成卡）

目标：围绕“定位 + 人设”产出可发布、可验证、可复用的优质选题。

输入：

- 定位一句话：<核心承诺>
- 主人设：<服务对象 + 方法风格>
- 目标受众：<主群体>
- 素材要点：<source 中的事实>
- 聚焦标签：<例如 增长飞轮,模板复用,案例拆解>
- 本轮目标：<例如 7 天产出 5 条>
- 选题总数：<例如 12 / 20 / 30>
- 渠道：<github/x/wechat...>
- CTA：<本轮主行动>

输出：

1. 按“选题总数”自动分配 P1/P2/P3（默认比例约 50% / 33% / 17%）
2. 输出 P1/P2/P3 三档选题清单
4. 每个选题附：
   - 标题
   - 切入角度
   - 内容形式
   - 开场钩子
   - 推荐渠道
   - CTA
5. 7 天发布节奏建议

硬约束：

- 每个选题都必须对应一个明确痛点。
- 每个选题都必须包含可执行动作或可验证证据。
- 不做“泛鸡汤”标题，不做夸张收益承诺。
